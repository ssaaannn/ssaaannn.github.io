Penjelasan Code CSS

  • 1. `body`: Mengatur gaya elemen body (seluruh halaman).
  • 2. `header`: Mengatur gaya elemen header.
  • 3. `main`: Mengatur gaya elemen main.
  • 4. `section`: Mengatur gaya elemen section.
  • 5. `ul`: Mengatur gaya elemen ul (unordered list).
  • 6. `li`: Mengatur gaya elemen li (list item).
  • 7. `footer`: Mengatur gaya elemen footer.
  • 8. `font-family`: Mengatur jenis font yang digunakan pada halaman.
  • 9. `line-height`: Mengatur tinggi garis dalam satu baris teks, untuk meningkatkan keterbacaan.
  • 10. `margin` dan `padding`: Mengatur margin dan padding untuk memberikan ruang antara elemen.
  • 11. `background-color`: Mengatur warna latar belakang elemen.
  • 12. `color`: Mengatur warna teks pada elemen.
  • 13. `text-align`: Mengatur perataan teks (center, left, right).
  • 14. `border-bottom`: Menambahkan garis bawah pada elemen.
  • 15. `max-width`: Menetapkan lebar maksimum untuk elemen.
  • 16. `@media screen and (max-width: 600px)`: Menerapkan gaya pada layar dengan lebar maksimum 600px, digunakan untuk membuat halaman responsif pada perangkat mobile.
