Penjelasan Code html

  • 1. <!DOCTYPE html> : Mendefinisikan jenis dokumen HTML sebagai HTML5.
  • 2. <html lang="en"> : Membuka elemen html dengan menentukan atribut bahasa (language) yang digunakan (dalam contoh ini adalah bahasa Inggris).
  • 3. <head> : Bagian kepala dokumen, berisi informasi-informasi meta dan tautan ke sumber eksternal seperti CSS dan JavaScript.
  • 4. <meta charset="UTF-8"> : Mendefinisikan pengkodean karakter dokumen sebagai UTF-8 (Unicode Transformation Format 8-bit), standar yang paling umum digunakan untuk karakter multibyte.
  • 5. <meta name="viewport" content="width=device-width, initial-scale=1.0"> : Mendefinisikan viewport untuk tampilan pada perangkat mobile, dengan lebar viewport mengikuti lebar perangkat dan rasio tampilan awal 1.0.
  • 6. <title>Peta Belajar Web Developer</title> : Menentukan judul dari halaman web, akan ditampilkan di tab browser atau saat halaman di-bookmark.
  • 7. <link rel="stylesheet" href="style.css"> : Menautkan file CSS eksternal (style.css) dengan halaman HTML.
  • 8. <body> : Bagian tubuh dokumen, berisi konten utama yang akan ditampilkan pada halaman web.
  • 9. <header> : Bagian kepala halaman, biasanya digunakan untuk menampilkan judul atau header lainnya.
  • 10. <h1>Peta Belajar Web Developer</h1> : Menampilkan judul utama halaman.
  • 11. <main> : Bagian utama konten halaman.
  • 12. <section> : Memulai sebuat bagian dari konten yang terpisah dari bagian lainnya (biasanya memiliki judul atau subjudul).
  • 13. <h2>Tahap 1: Dasar-dasar Web Development</h2> : Menampilkan subjudul atau judul bagian.
  • 14. <ul> : Memulai sebuah daftar tanpa urutan (unordered list).
  • 15. <li>HTML</li> : Menampilkan item dalam daftar.
  • 16. <footer> : Bagian kaki halaman, biasanya berisi informasi tambahan atau tautan penting.

Penjelasan Code CSS

  • 1. body : Mengatur gaya elemen body (seluruh halaman).
  • 2. header: Mengatur gaya elemen header.
  • 3. main: Mengatur gaya elemen main.
  • 4. section: Mengatur gaya elemen section.
  • 5. ul: Mengatur gaya elemen ul (unordered list).
  • 6. li: Mengatur gaya elemen li (list item).
  • 7. footer: Mengatur gaya elemen footer.
  • 8. font-family: Mengatur jenis font yang digunakan pada halaman.
  • 9. line-height: Mengatur tinggi garis dalam satu baris teks, untuk meningkatkan keterbacaan.
  • 10. margin dan padding: Mengatur margin dan padding untuk memberikan ruang antara elemen.
  • 11. background-color: Mengatur warna latar belakang elemen.
  • 12. color: Mengatur warna teks pada elemen.
  • 13. text-align: Mengatur perataan teks (center, left, right).
  • 14. border-bottom: Menambahkan garis bawah pada elemen.
  • 15. max-width: Menetapkan lebar maksimum untuk elemen.
  • 16. @media screen and (max-width: 600px): Menerapkan gaya pada layar dengan lebar maksimum 600px, digunakan untuk membuat halaman responsif pada perangkat mobile.
